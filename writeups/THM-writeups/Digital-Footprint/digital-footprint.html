Challenge Overview

Official Title: Digital Footprint - Beginner-friendly OSINT Challenge
Objective: Utilize OSINT techniques to investigate and locate "flag" through multiple digital investigation methods
CTF Type: Jeopardy-style with progressive discovery tasks

Updated Task Analysis

Task 1: Image Metadata Forensics

  • CTF Relevance: Exif data extraction challenge
  • Process:
    1. Download provided image file
    2. Use ExifTool to extract metadata: exiftool edited-house-1763031553617.jpg
    3. We will obtain map titles from the extracts.
    4. Clean and format location data: 26°12'14.8"S 28°02'50.3"E
    5. Search in mapping services with corrected coordinates
  • Flag Indicator: The city's name is flag.

Task 2: Archive Research

  • CTF Relevance: Historical data investigation
  • Process:
    1. Use provided address/coordinates
    2. Search on archive.org
    3. Search : `warc-acme.com
    4. Click on the displayed result and you get the Firstfiledate
  • Flag Verification: The Firstfiledate value is the flag.

Task 3: Reverse Image Intelligence

  • CTF Relevance: Visual intelligence gathering
  • Process:
    1. Upload challenge image to Google Images
    2. Identify location through reverse search
    3. Use Google Maps/Street View for virtual reconnaissance
    4. There is a phrase written above the small door on the side
    5. Translate this foreign phrase into English
  • Flag Confirmation: The flag is the name found at the building entrance on the map.

Task 4: Text File Analysis & Social Media Tracing

  • CTF Relevance: Multimedia intelligence correlation
  • Process:
    1. Reading the text file, we notice that it refers to a video
    2. Extract metadata from text file: exiftool internal-docs-1769695301727.odt
    3. Identify YouTube username from metadata
    4. visit YouTube channel URL
    5. We find a message with the flag written at the end of it
  • Final Flag: The flag is at the end of the message.
